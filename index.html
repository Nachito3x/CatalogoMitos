Introducción a HTML

HTML es el lenguaje de marcado utilizado para crear páginas web. Permite estructurar el contenido y definir su presentación.

Elementos básicos de HTML

  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Párrafos de texto.
  • <a>: Enlaces a otras páginas o recursos.
  • <img>: Imágenes.
  • <ul> y <ol>: Listas no ordenadas y ordenadas.
